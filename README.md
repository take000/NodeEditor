# リポジトリの取得
このコマンド使うかzipダウンロード
```
git clone https://github.com/take000/NodeEditor
```

# 環境構築バトル
<details>
  <summary>Windows</summary>
  
### venvを使う場合
```
cd NodeEditor
py -m venv venv
.\venv\Scripts\activate
```
### pipのアップデート
```
pip install --upgrade pip
```
### 依存ライブラリのインストール
```
pip install -r requirement.txt
```
</details>

<details>
  <summary>Linux,Mac</summary>
  
### venv使う場合
```
cd NodeEditor
python3 -m venv venv
source venv/bin/activate
```

### pipのアップデート
```
pip install --upgrade pip
```

### 依存ライブラリのインストール
```
pip install -r requirement.txt
```
</details>

# 実行
Windows
```
py main.py
```

Linux,Mac
```
python main.py
```
